Binary Divisibility By Five
medium
Given a binary number represented as a string bin, determine whether its decimal value is divisible by 5. Return the string "Yes" if it is divisible by 5, otherwise return "No".

You are to implement the function isDivisibleBy5() which takes bin as input and returns "Yes" or "No" depending on divisibility by 5. Aim for an algorithm with linear time in the length of bin and constant extra space.

The binary string contains only characters '0' and '1'. Leading zeros are allowed.

Input:

bin — a string representing a binary number.
Output:

A string: "Yes" if the decimal value of bin is divisible by 5, otherwise "No".
Expected Time Complexity: O(|S|)

Expected Auxiliary Space: O(1)

Constraints:

1 ≤ |S| ≤ 10^{5}
